# Flat Booking Website

A responsive flat booking web application developed using HTML, CSS, and Bootstrap. The project showcases modern UI components, image carousels, Google Maps integration, and booking modal functionality.

## Features

* Responsive flat listing layout
* Bootstrap image carousel
* Google Maps location embedding
* Booking modal form
* Modern card-based UI design
* Mobile-friendly interface

## Technologies Used

* HTML5
* CSS3
* Bootstrap 5

## Project Structure

```text
index.html
style.css
script.js
```

## How to Run

1. Download or clone the repository
2. Open the project folder in VS Code
3. Open `index.html`
4. Run using Live Server extension
